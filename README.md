# MIFIC Modern

Tema WordPress personalizado inspirado en el prototipo de Figma "Modernizar sitio web MIFIC".

## Instalación

1. Copia la carpeta `mific-modern` dentro de `wp-content/themes/` de tu instalación WordPress.
2. Entra al panel de WordPress.
3. Ve a `Apariencia > Temas`.
4. Activa `MIFIC Modern`.
5. Crea o asigna una página como portada desde `Ajustes > Lectura` para usar el diseño de `front-page.php`.

## Contenido incluido

- Header institucional con navegación, buscador y acceso de usuario.
- Hero azul con llamadas a trámites y servicios.
- Áreas de acción.
- Noticias y eventos.
- Trámites destacados.
- Métricas de impacto.
- Bloque de asistencia.
- Footer con enlaces rápidos, servicios, contacto y redes.
